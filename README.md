Веб-приложение для поиска фильмов по описанию и цитатам с использованием нейросетей.

Описание:
Приложение позволяет пользователю вводить описание сюжета, цитату из фильма или любую другую информацию, после чего нейросеть анализирует запрос, при необходимости обращаясь к веб-поиску, и возвращает результаты в виде карточек с постерами найденных фильмов.

Технологии:

- PHP — серверная часть
- PostgreSQL — база данных для хранения фильмов
- HTML, CSS, JavaScript — клиентская часть
- API OpenRouter.ai — доступ к нейросетевым моделям
- cURL — отправка запросов к API

Функциональность:

- Нейропоиск фильмов по описанию и цитатам
- Каталог всех доступных фильмов с постерами
- Выпадающий список подсказок при вводе
- Адаптивный интерфейс
- Логирование запросов

Установка и запуск:

1. Установить PostgreSQL и создать базу данных
2. Выполнить SQL-скрипт для создания таблицы movies
3. Настроить подключение к БД в файлах `api/search.php` и `api/getAllMovies.php`
4. Запустить встроенный веб-сервер PHP: php -S localhost:3000
